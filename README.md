## Overview

A portfolio collection of **10 vanilla JavaScript projects** — built without frameworks or libraries. Each project is a self-contained application focused on a specific concept, from consuming third-party APIs to working with native browser features and the DOM.

## Built With

- **HTML5** — semantic markup and structure
- **CSS3** — responsive layouts, transitions, and theming
- **Vanilla JavaScript (ES6+)** — application logic, DOM manipulation, and async/await
- **External APIs**
  - [OpenWeather API](https://openweathermap.org/api) — live weather data
  - [CoinGecko API](https://www.coingecko.com/en/api) — cryptocurrency prices
  - [QR Server API](https://goqr.me/api/) — QR code generation
  - [Web Speech API](https://developer.mozilla.org/en-US/docs/Web/API/Web_Speech_API) — text-to-speech synthesis

## Running Locally

```bash
# Clone the repository
git clone https://github.com/aliza-dev/js-projects.git

# Navigate into the project
cd js-projects

# Open any project in your browser
# (e.g. open the weather app)
open weather-app/index.html
```

No build step or dependencies required — each project runs directly in the browser.

## What I Learned

- Consuming and handling third-party REST APIs with `fetch` and `async/await`
- Manipulating the DOM and managing application state without a framework
- Persisting user data across sessions using `localStorage`
- Working with native browser APIs such as the Web Speech API and HTML5 Drag and Drop
- Writing clean, modular, and maintainable vanilla JavaScript (ES6+)
